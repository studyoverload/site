如果重來，我會如何準備學測 (如何設定目標)

1 Comment / 想進步

每年五六月會有滿多學長姐回到高中，向升高三的同學分享學測的準備心得。藉這個機會，我想聊聊當初我是怎麼準備，有哪些我希望能做得更好。這一系列心得會分成不同面向討論，第一篇是關於如何設定目標。

為什麼從目標設定談起？

回想當年，每個人都忙著向學長姐請教如何提升讀書效率、如何調適心情時，很少人會談談考完試之後會發生什麼事。進大學、大學畢業後、找工作有什麼標準可以衡量嗎？準備考試的時候，我們是為了什麼而做？我們從這一整個過程裡學到了什麼可以應用在往後的人生？

考生如我沒想這麼多，每天過規律的作息：寫題目、整理筆記、累了小睡、放學跑操場。直到考完試，有學校念了，完成階段性的任務後，才開始想「然後呢？」(What now?)

好像這一切，都是且走且看，難道沒有什麼更深層的意涵嗎？ 缺乏了大局觀思維 (Big Picture Thinking)，我頓時迷茫了起來，意識到「目標」的重要性。

如何設定目標？

那要怎麼找到目標呢？

目標可大可小，大至「大學畢業後想做什麼職業、選什麼校系」，小至「每天安排的讀書進度」。

選什麼職業 ➡️ 對高中生來說太遙遠，很難說以後會做什麽樣的工作。

選什麼校系 ➡️ 看個人志向和最後分數夠不夠，並非完全在自己的掌握之中。

每天安排的讀書進度 ➡️ 看自己對整個學測的準備規劃而定。

要設定合適的目標要考量的事情太多了，就是要從哪裡著手才好？我需要一個立足點。

1️⃣ 從認識自己開始

老話一句，做任何決定前都要先了解自己。目標設定是非常個人的事情，儘管你收集了許多資料、觀察同學們都做了什麼，終究還是回歸到自己。不論我在這篇文章說了什麼，先沈澱一下，不要照單全收。（對沒錯，這是免責說明 😂）

認識自己是一生的課題。很多時候你覺得了解自己，但在某些情況下又發現好像不是那麼一回事。對於迷惘，可以拿一張紙，畫出四個象限如下：

這是周哈里窗理論（Johari Window），可以應用於盤點自己的個性、價值觀、喜歡/不喜歡，擅長/不擅長的事。哪些是大家都知道的，又有哪些是不為人知的，將它填上去。

接著請你的家人、朋友、師長給予回饋，了解你在他人的眼中是什麼形象，將原本不知道的特質從「盲目我」拉近「開放我」的象限。藉由這個機會，校準自己的定位。你希望呈現什麼樣的自己在他人面前？目前你的形象和想像中有什麼差距？

最後是「未知我」，這需要不斷地探索、嘗試新事物，面臨新挑戰的時候，才會知道有什麼等著你發覺，能讓你說出「原來我會選擇 xx ，犧牲了 xx ，有 xx 的優勢為我帶來新機會，這是 x 年前的我完全意想不到的事情呀！」

 

在我高中的時候，有 2 個例子分享：

參加社團活動，在團體裡擔任不同的角色：未知我 ➡ 開放我（和大家一起探索學習）

在成果發表會的時候，在各個表演項目間需要有故事小劇場穿插（串場），在等待下一個節目開始前不會冷場。

當時我就負責寫劇本，編了幾個劇情，既搞笑又能與節目有所連結。團隊夥伴的反應都滿好的，這就讓我了解到自己喜歡寫作，而且能夠配合別人的需求，產出相對不錯的成果。

老師請同學們寫互評表：盲目我 ➡ 開放我（別人對自己的評價）

我的導師給每個人 10 張互評表，請我們拿給比較要好的同學填寫。裡面包含三個部分：優點、改進會更好、悄悄話。

大家的回饋都滿認真寫，分享了一些平常不太會聊到的想法。這讓我收穫良多，有些特質是我之前沒有注意到的

如果沒有互評表這個契機，平常也可以在聊天的時候，主動問問同學對你有什麼想法。問的時候保持自然，不用是念問題清單的感覺。主動讓對方知道這些問題的脈絡和你的想法。如果對方不太想回答，就不要為難彼此 😂

 

2️⃣ 以終為始

在《與成功有約》（The 7 Habits of Highly Effective People），書中所提出的「高效人士的 7 個習慣」第二項是「以終為始」(begin with the end in mind)。當你能夠為自己的人生負責，主動積極面對不同的處境後（Proactive)，下一步就是先構思你的目標，再開始執行。

先釐清自己的價值觀，再安排你的優先次序。

  • 你的理想生活是什麼樣子？你目前要怎麼做才能達到目標？
  • 別人是如何達到你想要的生活？
  • 你要什麼？不要什麼？
  • 有其他什麼方法，可以幫助你達到目標呢？

當你列出這些問題的時候，不妨給自己一段時間想想，盡量越具體越好，依照結果看有沒有能夠改善的地方。例如：

  • 我想過著清閒，不用整天操心的生活 -> 所以 工作/理財/家庭規劃是…
  • 別人透過 xx 的職涯規劃，提早退休/上挑選適合伴侶的系統化課程，有了開闊的感情觀…
  • 想要在親近自然的地方/不想要在充滿競爭的環境…
  • 達到目標的方式：先想清楚什麼是我要的「清閒」、如何照顧我的心理健康…

「以終為始」常見的問題是：那我怎麼知道我的「終」是不是正確的？萬一我不想要原先設定的目標怎麼辦？

歷練和眼界有所成長後，修正目標也沒關係，不用糾結於對錯。從高中到現在，很多我覺得「事情應該要怎麼樣」都不斷被打破，對於自己的認識也一直更新。執行的過程中，會發現牽一髮動全身，而且很多因素都不是自己能夠掌控的。有時候，看似很難從目前的狀態找到通往「終」的道路，就需要其他方法輔助。

 

如何朝向目標前進？

要達到目標，試著將定義抽象的概念，拆成數個小問題問自己。舉例：

目標：要成為英文很好，能和外國人對答如流的人。

  • 什麼是英文很好？
  • 重視口說的什麼部分？
  • 聊什麼？日常英文還是商用英文？
  • 如何讓雙方都享受對話？

逐一拆解這些問題，把它寫成計畫表，每天分配一點任務，累積成可量化的實力。這時你可以運用 PDCA 循環來管理計畫。

計劃 Plan ➡️ 執行 Do ➡️ 覆盤 Check ➡️ 行動 Action

例如：

Plan 建立自己的部落格，目的是分享給有需要的人、練習文筆和記錄生活，每一季撰寫不同的主題的學習筆記。
Do 經過一季的執行後，紀錄發表文章數、寫作時數、研究心得等。
Check 發現有些主題不是三個月就能完成的，它需要時間的累積和心思，才能研究出心得。網站造訪次數不如預期。
Action 觀察別人如何經營部落格、對該主題的學習方式，詢問讀者感受，檢視自己的內容可以如何修正。

經過這四個步驟的不斷循環，來優化你的程序和結果，來達到最終的目標。要達到目標，最重要的是持之以恆，營造讓你想要繼續做的環境。

既然是長久的計畫，建立習慣能促使你繼續下去。前陣子廣受歡迎的 原子習慣 提到要建立好習慣並持續進步的方法：

讓提示顯而易見，讓習慣有吸引力，讓行動輕而易舉，讓獎賞令人滿足。

接續上一個例子，如果使用原子習慣的方法：

提示 桌上只放電腦，保持整潔。將 .md 檔（寫部落格文章的頁面）保持開啟狀態，關閉其他娛樂的網頁。每天固定在起床後的一小時，天氣清爽時撰寫文章。
渴望 多逛逛其他人的部落格，吸收新知識，想像自己有一天成為他們的樣子。
回應 告訴自己每天寫一小時就好。偶爾寫到一個程度會想要一直寫下去。閱讀的時候，也會有意識地收集素材。
獎賞 每次寫完一篇文章後，會休息一天，做點別的事情切換心情。通常是豐盛的早餐搭配放空時間 😌 如果想要偷懶的話，跟自己說，「那你連續 xx 天寫作的紀錄就會終止了耶 😣」

這聽起來不難，不過停滯期往往需要花比想像中更長的時間。在《蝙蝠俠：黑暗騎士》中有一句話「黎明前是最黑暗的」，要達到目標，需要耐心和毅力，也需要十足的勇氣。能夠在夜色漸深中，懷抱著信念繼續往前走，最終堅持到黎明升起。在這之中，有多少人是中途就放棄？或是走到別條路的？將實現目標的過程就是一場耐力賽，持續思考、行動、改善才能走得長遠。

給高中生的話

謝謝你讀到這裡 😊 17、18歲都還很年輕，沒有明確的方向很正常，但「開始想想自己的目標並著手進行」這個概念是我希望在高中能夠學到的。

這篇文章的標題是「如何準備學測」，可能很多同學在高一高二就以「考好學測」為目標，過著被讀書和考試充斥的生活。但高中不僅是準備考試的三年，更是學習擴展視野的機會。藉由參加社團，結交新朋友，發展對不同事物的興趣，來更認識自己。在嘗試的過程中，可能會找到你的熱情所在，也有可能遇到挫折。好玩之處在於沒人知道會發生什麼事 😅 這些都沒關係，過一段時間回來看，會覺得這些都是很寶貴的回憶，甚至為此寫文章分享給之後的高中生 😌

在我高中的時候，我的成績排名一般，對未來要做什麼工作也沒有明確目標，倒是花了不少時間看美劇，參加社團活動。在做這些看似「休閒」對學業沒直接幫助的事情時，我發現自己對於外語滿有興趣，就算聽不太懂電視劇的對話，也會想要花時間去查、模仿他們說話的樣子。遇到外國人的時候，也不太會緊張，能夠保持平常心應答。後來做的很多決定，源自於這段時間的探索。發掘讓自己樂此不疲的事物，以此延伸至更宏觀的目標，慢慢累積實力。很慶幸高中的時候有多一點行動更認識自己 😎

很多人覺得既然要設定目標，就是要看起來難以達到，或是聽起來很厲害，才叫做「好目標」。你也可以從自身出發，開始做些什麼，再走出更清晰的路線，自然會知道接下來要做些什麼，不需要拘泥於設定所謂「宏遠的目標」。

 

結語

雖然標題是寫給高中生的，這些目標設定的方法仍然適用於成人的生活。

將目標拆解成小步驟，每天一點一滴累積成果。利用 原子習慣 講述的方法來建立習慣，對我來說很有幫助 😎 每天進步 1%，一年後，會比現在進步 37 倍。聽起來相當誘人！一年後，再回來分享我進步了多少，一起加油 🤜🤛
